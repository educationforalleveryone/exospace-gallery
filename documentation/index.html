Exospace 3D Gallery

Thank you for purchasing Exospace 3D Gallery. This application allows you to create immersive, virtual 3D art exhibitions directly from your browser. No coding knowledge is required.

Version: 1.1.0
Release Date: January 2026

Server Requirements

Exospace requires a standard LAMP/LEMP stack. Most shared hosting providers (Namecheap, Bluehost, SiteGround) support this out of the box.

  • PHP 8.2 or higher
  • MySQL 8.0+ or MariaDB 10.3+
  • Apache or Nginx Web Server
  • BCMath, Ctype, Fileinfo, JSON, Mbstring, OpenSSL, PDO, Tokenizer, XML, GD extensions

Installation Guide

We have included an automated installer to make setup easy.

1

Upload Files

Unzip the download package and upload all files inside the exospace folder to your server's public_html (or subdomain) folder.

2

Set Permissions

Ensure the following folders are writable (775):

  • storage/ (and all subfolders)
  • bootstrap/cache/
3

Run Installer

Open your browser and navigate to your website URL (e.g., https://yourdomain.com/install).

Follow the on-screen wizard:

  1. Check Requirements
  2. Enter Database Credentials
  3. Create Admin Account & Enter Purchase Code
Important: The installer will automatically create the .env file and delete itself upon completion for security.

Creating Your First Gallery

Once installed, log in to the Admin Panel.

1. Create Gallery

Click + New Gallery on the dashboard. Enter a title and choose your style settings (Wall Texture, Floor Material, Lighting).

2. Upload Images

After creating the gallery, you will be redirected to the Edit page. Drag and drop your artwork images into the upload zone.

  • Max file size: 5MB per image
  • Max images per gallery: 100 images
  • Supported formats: JPG, PNG, WEBP
  • Recommended resolution: 1920px width for best quality

3. Manage Images

You can manage your uploaded images efficiently:

  • Delete single image: Click the red X button on any image
  • Batch delete: Select multiple images using checkboxes, then click "Delete Selected"
  • Bulk upload: Drop up to 100 images at once - the system will process them automatically

4. View & Share

Click the View button on the gallery card to launch the 3D experience. Copy the URL to share it with the world.

Advanced Customization

Developers can customize the 3D engine by editing resources/views/gallery/view.blade.php.

Adding New Textures

Upload seamless texture files to public/assets/textures/walls/ and update the TEXTURE_PATHS constant in the JavaScript code.
